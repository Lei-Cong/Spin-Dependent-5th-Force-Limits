# SD5thF brand assets

These vector files are ready for download and retain their quality at any size.

| File | Recommended use |
| --- | --- |
| `sd5thf-primary.svg` | Primary horizontal identity on dark backgrounds |
| `sd5thf-reverse.svg` | Horizontal identity on light backgrounds |
| `sd5thf-compact.svg` | Compact horizontal spaces |
| `sd5thf-grid.svg` | Square mark, dark version; avatars, posters and social media |
| `sd5thf-grid-light.svg` | Square mark for light layouts and print |
| `sd5thf-brand-sheet.svg` | Overview and presentation sheet |

Please preserve the proportions, colors and spacing of the artwork. The four colored rules refer collectively to the four known fundamental interactions; they are a visual spectrum rather than a one-to-one label for the four characters.
