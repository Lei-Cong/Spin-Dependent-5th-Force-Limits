# SD5thF logos

Official vector identity files for the SD5thF Research & Community Platform. Click any SVG on GitHub to preview or download it.

| File | Use |
| --- | --- |
| `sd5thf-primary.svg` | Primary horizontal logo for dark backgrounds |
| `sd5thf-reverse.svg` | Horizontal logo for light backgrounds and print |
| `sd5thf-compact.svg` | Compact horizontal spaces |
| `sd5thf-grid.svg` | Square mark for dark presentation contexts |
| `sd5thf-grid-light.svg` | Square mark for light layouts, avatars, and print |
| `sd5thf-brand-sheet.svg` | Brand overview and presentation sheet |

Please preserve the proportions, spacing, and colors. The four colored line segments refer collectively to the four known fundamental interactions; they are a visual spectrum rather than a one-to-one label for the four characters.
